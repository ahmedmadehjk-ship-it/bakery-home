🎂 دليل رفع Bakary Home على الإنترنت

اتبع الخطوات دي بالترتيب وموقعك هيبقى أونلاين في أقل من 15 دقيقة

⚡ اللي هتعمله في الجملة

GitHub (تخزين الكود) → Railway (تشغيل السيرفر) → رابط موقعك الرسمي ✅

١
إنشاء حساب GitHub ورفع الكود
1

افتح github.com

اضغط Sign up وعمل حساب بإيميلك

2

عمل Repository جديد

بعد الدخول، اضغط الزرار الأخضر "New" أو "+" في الأعلى

3

سمّي الـ Repository

في خانة Repository name اكتب: bakary-home

اختار Public واضغط Create repository

4

رفع الملفات

في الصفحة اللي هتظهر، اضغط "uploading an existing file"

افتح الزيب بتاعك وارفع كل الملفات والفولدرات جوا فولدر bakary-node

مهم: ارفع محتوى الفولدر مش الفولدر نفسه — يعني server.js وpublic وpackage.json مباشرة
5

احفظ الملفات

تحت الصفحة اضغط "Commit changes" الزرار الأخضر

٢
رفع الموقع على Railway
1

افتح railway.app

اضغط "Login" ثم "Login with GitHub" — هيستخدم نفس حساب GitHub بتاعك

2

عمل مشروع جديد

اضغط "New Project" ثم "Deploy from GitHub repo"

3

اختار الـ Repository

هتلاقي bakary-home في القائمة — اضغط عليه

لو مش ظاهر، اضغط "Configure GitHub App" وادي Railway صلاحية وصوله
4

اضغط Deploy

Railway هيشتغل تلقائي وهيبني المشروع — استنى دقيقتين لحد ما يخلص

5

جيب الرابط بتاعك

بعد ما يخلص، اضغط على المشروع ← تبويب Settings ← Networking ← "Generate Domain"

هتاخد رابط زي: bakary-home.up.railway.app ✅

٣
تأكيد إن كل حاجة شغالة
1

افتح الموقع

افتح الرابط بتاعك في المتصفح وتأكد إن الموقع بيظهر

2

جرب الفورم

اكتب طلب تجريبي في صفحة "تواصل معنا" واضغط إرسال

3

افتح لوحة الأدمن

افتح: رابطك/admin.html

كلمة السر: bakery2025
